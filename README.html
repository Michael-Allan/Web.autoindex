Web Autoindex: presenting indeces of directory content

Here is project Web Autoindex. This project houses files to configure the presentation of indeces of directory content by the `mod_autoindex` module of Apache HTTP Server. For a usage example, see:

  • `5_autoindex.conf`
  • `autoindex/readme.conf`
  • `autoindex/Git_Hub/readme.conf`
